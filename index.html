Interactive Visualization Using the Building Inventory Dataset

Below is an example of an interactive visualization using vega-lite. The first graph shows the total square footage acquired each year. The second graph is a barchart showing the total square footage acquired by each agency

The interativity is seen when a certain year is selected in the chart one, chart 2 will highlight the specific agencies that acquired buildings (square footage) in that given year. The selected agencies will be highlighted in an orange color.

I think this is a very interesting interactive visualization.
